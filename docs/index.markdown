# IPN Winter School

This is the coding-along webpage for the IPN Winter School workshop **“A (Gentle) Introduction to Natural Language Processing”**. It provides a set of exercises that delve deeper into the main concepts and ideas discussed in each module, as well as hands-on experience in building and running your own NLP pipelines. All code is self-contained and available as Google Colab notebooks.
